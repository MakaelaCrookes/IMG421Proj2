# IMG421Proj2
Apple Tree Picker Game with Easy, Medium, and Hard modes.

## Instructions
1. Use mouse to move the basket and catch apples
2. Avoid catching bombs in your basket
3. If you miss an apple or catch a bomb you will lose a life

<hr>

### Easy Mode:
In easy mode there is slower tree movement, less direction changes, and 5 lives.
### Medium Mode:
In medium mode the tree moves faster, changes directions more fequently, and apples/bombs will spawn in more frequently. You have 3 lives in this mode.
### Hard Mode:
In hard mode you have the same things as medium mode but must play the game upside down with only 2 lives.
